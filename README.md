# Application E-commerce – React & FakeStoreAPI

##  Description
Cette application est un projet pédagogique développé avec **React** afin de consommer l’API **FakeStoreAPI**.  
Elle simule une plateforme e-commerce avec authentification, panier et suivi des commandes.

---

## Fonctionnalités
- Authentification utilisateur
- Affichage des produits
- Détails d’un produit
- Gestion du panier
- Suivi et historique des commandes
- Interface responsive

---

## Technologies utilisées
- React
- React Router
- Context API
- Axios
- FakeStoreAPI
- Git & GitHub
